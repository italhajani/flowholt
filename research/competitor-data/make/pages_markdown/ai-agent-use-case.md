# AI agent use case

1 min

This page reflects a **previous version of Make AI Agents**. For the latest information, see [Make AI Agents (New)](/make-ai-agents-new)﻿.

In the following use case, we will build an agent that helps us manage the stock inventory of a small shop.

We will use Slack to submit tasks for the agent and receive responses about their completion. The agent will use scenarios﻿ and other information we provide to him to list our stock information and order new items.

In this use case, we will:

1

﻿[Set up the AI agent](/step-1-set-up-the-ai-agent)﻿

2

﻿[Create the AI agent's tools](/step-2-create-the-ai-agents-tools)﻿:

* Scenario to list shop inventory

* Scenario to order more stock

3

﻿[Create a scenario to send tools to the AI agent](/step-3-create-a-scenario-to-send-tasks-to-the-ai-agent)﻿

**Prerequisites**:

* API key of the AI service provider. Check the documentation of your AI service provider to find out how to get it.
