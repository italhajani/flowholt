# Step 1. Set up the AI agent

1 min

This page reflects a **previous version of Make AI Agents**. For the latest information, see [Make AI Agents (New)](/make-ai-agents-new)﻿.

First, we need to create the AI agent and connect to an AI provider. We can do that in the **AI Agents** configuration tab.

1

Navigate to the **AI Agents** configuration tab in the left sidebar, then click the **Create agent** button.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/5cWtnw_pDo25ztrA6JmFk-20250929-073114.png "Document image")

﻿

Alternatively, you can create a new AI agent in the **Make AI Agents** > **Run an agent** module. In the module settings, click the **Create agent** button.

2

In **Connection**, select your AI provider connection.

Users on **all plans** can select **Make's AI Provider**. Those on **paid plans** can also choose a **custom AI provider connection**, such as OpenAI or Anthropic Claude.

1. If you don't have a connection to an AI provider, click **Add** to open a **Create a connection** dialog.

2. In the **Connection type** dropdown, select your AI provider.

3. If applicable, fill in the rest of the form according to the documentation for creating a connection to the AI provider. Check the list of [AI providers](https://www.make.com/en/help/ai-in-make "AI providers").

4. Click **Save** to complete the connection.

3

In the **Agent name** field, specify the name of the agent.

4

In the **Model** field, select the AI model that the agent will use for reasoning.

5

In the **System prompt** field, define the agent's purpose and constraints. For our shop inventory agent, fill in: "You are a stock inventory bot."

6

Click **Save** to confirm.

You have created an AI agent for managing your stock inventory. However, your agent doesn't have any tools to do its job. In the next section, we'll create scenarios﻿ that function as tools for the agent.
