# Rebrand your instance

The system settings page lets you customize the appearance, name, support, and available features of your instance. To access and edit these settings, go to Administration > System Settings and find the corresponding field.

### hashtag Name and appearance

The following fields let you customize the name and appearance of your instance.

Brand title

The name that appears in the browser tab and in email communication to your end user.

Brand theme

Select one of the available colors. This color applies to the login page, public side navigation bar, and other UI features.

Brand favicon

Upload or drag and drop a .ico file to appear in browser tabs when your users have it open.

```
.ico
```

Brand logo (for dark background)

Upload or drag and drop a .png file to appear against dark backgrounds.

Recommended size is 512 x 512 px with a square image.

The logo you upload here appears on the login screen.

Brand logo (for light background)

Upload or drag and drop a .png file to appear against light backgrounds.

Recommended size is 512 x 512 px with a square image.

The name and appearance settings do not impact the administration interface. You need to click Leave administration to confirm the changes. Recent changes may take several minutes to appear in your UI.

### hashtag Support root URLs and emails

Support root URLs provide a base URL for UI links to support documentation and tickets.

Support email

The name and appearance settings do not impact the administration interface. You need to click Leave administration to confirm the changes. Recent changes may take several minutes to appear in your UI.

SDK App Help Root

This URL only impacts the custom app editor and is not necessary if the Apps Platform is disabled. Enter the URL for documentation on creating custom apps.

Help Center Root

This URL provides a base URL for in-product user support. Your users encounter this when they click on the ? when editing module settings.

Open Ticket Root

This URL links to your customer support ticketing system. When users click on Contact Support, a new tab opens with the URL you enter in this field.
