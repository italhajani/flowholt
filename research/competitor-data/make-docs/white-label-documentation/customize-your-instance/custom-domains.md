# Custom domains

Make's custom domains support the following:

- Public and admin interfaces are both available on your custom domain
- Emails from the custom domain to your end customers
- OAuth redirects with the custom domain
- Base and Primary domains for your instance
- API calls, webhooks, and mailhooks using any configured domain
