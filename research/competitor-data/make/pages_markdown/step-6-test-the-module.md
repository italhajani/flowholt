# Step 6. Test the module

1 min

Testing makes sure your module is working correctly before you continue building your scenario.

To test the **Google Sheets > Watch New Rows** module:

1

Right-click the **Google Sheets > Watch New Rows** module.

2

Select **Run this module only.**

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/HQBmTqLdhFw4tnWL-dNgN-20251103-195155.png?format=webp "Document image")

﻿

When the run completes, an output bubble will appear above the module.

3

Click this bubble to view the results. Since this module reads rows from your **Prospects** spreadsheet according to the limit you set in [Step 5](/step-5-set-up-the-trigger)﻿, you'll see all your spreadsheet data when you click the output bubble.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/bnImgZ1C4qIf8WjR0vEKx-20251103-195219.png?format=webp "Document image")

﻿

4

Compare the data in the operation output with your original spreadsheet to verify that all information appears correctly.

![Document image](https://images.archbee.com/yjwhINLS3fc-NXg38fV_d-VXMppkIArP5-JCPsbStlC-20250321-171344.png?format=webp "Document image")

﻿

When creating a scenario, regularly test each step to identify potential issues early.

Every test run for this specific module consumes one [credit](https://help.make.com/credits#CHMs2 "credit") from your account.
