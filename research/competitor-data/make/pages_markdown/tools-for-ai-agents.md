# Tools for AI agents

2 min

This page reflects a **previous version of Make AI Agents**. For the latest information, see [Make AI Agents (New)](/make-ai-agents-new)﻿.

In Make﻿, you can add three types of tools for AI agents: **module tools, scenarios,**  and **MCP tools****.**

* ﻿[**Module tools**](https://help.make.com/module-tools-for-ai-agents "Module tools") are best for a simple and quick setup. They let you add single module(s) to your AI agent without having to create and configure a full scenario﻿.

* ﻿[**Scenarios**](https://help.make.com/scenarios-for-ai-agents "Scenarios") are better for complex workflows where you need multiple steps, filters, and specific [scenario inputs and outputs](https://help.make.com/scenario-inputs-and-outputs "scenario inputs and outputs") in your AI agent.

* ﻿[**MCP**](https://help.make.com/mcp-tools-for-ai-agents "MCP")servers let your AI agents access new tools and perform tasks using them.

Key difference between these three tools:

**Module tools**

To add it to your AI agent, you just need to create a module tool and configure the module settings.

A scenario for a module tool with all the scenario inputs and outputs is created automatically. A scenario consists of the selected module and may contain 2-3 utility modules necessary to work with scenario inputs and outputs, and aggregation of multiple bundles for a search module type.

**Scenarios**

To add it to your AI agent, you have to manually build the scenario, add the module(s) you want to use, and define and map the scenario inputs/outputs.

A scenario may contain different modules and filters.

**MCP**

To add it to your AI agent, you need to connect to an MCP server and add the required tools to your AI agent.
