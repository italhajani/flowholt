# Make now a built-in connector on Anthropic Claude

2 min

Make is now a built-in connector on Anthropic Claude, making it faster and easier than ever to connect [Make MCP server](/make-mcp-server)﻿ to Claude.

Previously, connecting required manual steps, such as adding a URL and token. Now, find Make from the available connectors and complete the OAuth flow—no extra configuration needed.

![Document image](https://archbee-image-uploads.s3.amazonaws.com/oAyFj2GHlBeBVWF5OAir2/w8qa1DP_0H8rQoqbQ0KQ8-20260324-132256.gif "Document image")

﻿

## Key benefits

* **Simpler connection setup**: Add Make MCP server to your Claude account in a few clicks, without adding connection URLs or tokens.

* **Smarter scenario runs**: Send requests in the chat, then let Claude choose the right scenario for the task.

* **Run scenarios from Claude**: Trigger any scenario for any task without signing into Make.

## How to get started

To use Make from Claude, connect Make MCP server to Claude:

1

Open your Claude account.

2

Click your profile name on the left sidebar, then **Settings**.

3

Go to **Connectors**.

4

Click **Browse connectors**.

5

Search for Make and click the plus sign.

6

In the OAuth consent screen, select your organizationand its scopes.

7

Click **Allow** to proceed.

You've connected Make MCP server to Claude using the new Make built-in connector.

To learn about other connection options, see [Make MCP server](https://developers.make.com/mcp-server "Make MCP server").
