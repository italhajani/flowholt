# Scenario blueprints

7 min

In Make﻿, you can save or copy scenarios﻿ as **Blueprints** and share them with other Make﻿ users within or outside your organization, or use and reuse them for your needs.

A Blueprint is a reusable version of a scenario﻿ that includes:

* ﻿Scenario﻿'s modules

* Module settings

* Mapped values

Blueprints help to set up automations quickly, without having to start from scratch. You can:

* Back up your scenarios﻿ in case you lose access to or switch to another Make﻿ account.

* Export and save a blueprint to reuse a scenario﻿ later.

* Share a blueprint with other users, who can then import the file or paste the JSON and run the scenario﻿ as-is or adapt it to their specific needs.

* Copy a blueprint and paste it into other scenarios in Make, LLMs, or other relevant platforms.

While Blueprints include scenario﻿'s modules, settings, and mapped values, users still need to [create connections](https://help.make.com/connect-an-application#QzWef "create connections") for their accounts after importing the blueprint.

## **Export a blueprint**

You can export a blueprint of the scenario﻿ as a .json file.

To export a blueprint:

1

Click Scenarios﻿ in the left sidebar. Choose an existing scenario﻿, or click **+ Create a new scenario**, if you need to create the scenario﻿ first.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/d6pT0Xe-ljGjZ9nXiyx6I-20251001-152632.png?format=webp "Document image")

﻿

2

If it's a new scenario﻿, add modules and [configure the module settings](https://help.make.com/module-settings "configure the "). If it's an existing scenario, check that all of them are set up correctly.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/_GISVgX7C5_VfSraVGi7F-20251001-152717.png?format=webp "Document image")

﻿

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/DUb4kvqu6f__ZEwyj-G1E-20251001-152747.png?format=webp "Document image")

﻿

3

In the upper right corner of the Scenario Builder, click the **three dots** **>** **Export blueprint**.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/IS7LhhQ1Ka8NmOxWiiKpG-20251104-114924.png "Document image")

﻿

Once exported, the blueprint downloads to your device as a .json file.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/5qi2Ue5_Ah6ilGn3ggk2n-20251015-102626.png "Document image")

﻿

The file is typically named after your scenario﻿. For example:

* If your scenario﻿'s name is "New prospect notification", the file will be called New prospect notifcation.blueprint.json.

* If the scenario﻿ has a default "New scenario" name, Make names the file after the modules in the scenario﻿, e.g.,Integration Google Forms, Google Sheets.blueprint.json.

![Document image](https://app.archbee.com/api/optimize/PL8X94efBsjvhfQV3wyyj-qzh9fMSCOPod8wSEybfwD-20250630-134806.png "Document image")

﻿

You can now share this file with other Make users or keep it for your own future use.

## Import a blueprint

We recommend creating a new scenario﻿ and importing the blueprint there.

If you import a blueprint while editing another scenario﻿ that hasn't been saved, all the changes in that scenario﻿ will be lost.

You can import a blueprint of the scenario﻿ saved in .json format.

To import a blueprint:

1

In the upper-right corner of the Scenario Builder, click the **three dots** **>** **Import blueprint**.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/D3T2oUnqNbgfugMjsT7Bk-20251104-115541.png "Document image")

﻿

2

Click **Choose File,** select the blueprint file in .json format, and click **Save**.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/YH7zE6xK0j0SWorvT9S-M-20251001-152948.png "Document image")

﻿

The imported scenario﻿ will appear in the Scenario﻿ Builder. You can now configure your own account connections, make any necessary edits to the scenario﻿, schedule it, and click the **Save** icon.

The imported blueprint must be below 2 MB in size. If the blueprint exceeds the scenario size limit, you'll be unable to save the scenario and will see an error message.

## Copy a blueprint to a clipboard

You can also copy a scenario﻿ blueprint to the clipboard. The blueprint is copied as JSON text, which can be pasted into other scenarios in Make, LLMs, or other compatible tools.

To copy the blueprint to the clipboard, click the **three dots** **>** **C****opy blueprint to clipboard** in the upper-right corner of the Scenario Builder.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/w5mazxZRzNW7MMUwc4bg9-20251104-135955.png "Document image")

﻿

To paste the blueprint into another scenario in Make:

1

Create a new scenario or open an existing one.

2

Right-click anywhere on the scenario canvas.

3

Click **Paste**.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/nI8GjI-VF3VaW2w2JPFnv-20251104-140702.png "Document image")

﻿

A scenario from the blueprint will appear in the Scenario Builder. You can now configure it, make any necessary edits to the scenario﻿, schedule it, and click the **Save** icon.
