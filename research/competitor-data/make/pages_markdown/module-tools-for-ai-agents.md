# Module tools for AI agents

9 min

This page reflects a **previous version of Make AI Agents**. For the latest information, see [Make AI Agents (New)](/make-ai-agents-new)﻿.

When you use a module as a tool for an AI agent, you don't need to create or configure a scenario﻿ manually. Each module tool automatically creates its own scenario﻿ with all the necessary inputs and outputs*.* Just configure the module tool the same way you'd configure the module itself.

You can review the history and execution logs for the module tools' scenario﻿ at any time.

Each module tool is considered an active on-demand scenario. If you’re on the **Free** plan, remember that you can only have two active scenarios at a time.

Create a module tool

You can create a module tool in the **AI Agents** configuration tab or in the **Make AI Agents > Run an agent** module.

To create a module tool in the **AI Agents** configuration tab:

1

Navigate to the **AI Agents** configuration tab in the left sidebar.

2

Select an agent you want to add the module tool to.

If you don't have an agent yet, refer to the [Create agents](https://help.make.com/manage-ai-agents#Ln8Sl "Create agents") section.

3

In the **Tools** section, click **Add**, and select **Module**.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/QEeUh8kbyW-QNJXElGCQh-20250923-074317.png "Document image")

﻿

4

Click **Select a module tool**, and then **Create module tool**.

![Document image](https://app.archbee.com/api/optimize/PL8X94efBsjvhfQV3wyyj-iXE3hukRRKLEF4JNr23vQ-20250822-105319.png "Document image")

﻿

5

Select the app and then the module you need.

![Document image](https://images.archbee.com/PL8X94efBsjvhfQV3wyyj-DhtD3sk4_bYRmZa7SF3EQ-20250822-105909.png?format=webp "Document image")

﻿

The modules available in the app search for module tools include verified apps, excluding trigger modules, Flow Control, and Tools modules.

6

Configure the module tool's settings:

* Optional: In the **Name** and **Description** field, change the default name and description of the module tool. Clear and precise data helps an AI agent deliver better results.

* If required, create a new connection or select an existing one.

* Configure the module settings relevant to the chosen module.

For the fields related to scenario﻿ inputs and outputs, you can check the **Let AI Agent decide** box to have the AI agent fill them in automatically.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/1NcKlrBy2Wf_OXam1jJ1K-20250926-161854.png "Document image")

﻿

7

Click **Add** to add the module to **Tools**.

Once you create a module tool, you can use it in your AI agent. It appears in the **Tools** section of your AI agent.

In the **AI Agents** configuration tab, you can test the module tool by typing a request in the **Testing & Training** chat.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/q4rs-GqdN2Fe4ygiEYz-Y-20250929-074804.png "Document image")

﻿

You can also create a module tool in the **Make AI Agents > Run an agent** module:

1

Open the **Make AI Agents > Run an agent** module in a scenario﻿.

2

In the **Agent** field, select your agent**.**

3

Click **Add tools > Module**.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/j7oQpFXtLF0wrjdn1suO8-20250924-080442.png "Document image")

﻿

4

Follow the same steps as when creating a module tool in the **AI Agents** configuration tab.

### Scenarios for module tools

Every time you create a module tool, a new scenario﻿ is automatically created for it. To view it, click the expand icon next to the module tool name.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/JPyNtvhhZDgVyXY6hhwbH-20250924-080741.png "Document image")

﻿

A scenario created for a module tool is locked by default. To edit it, click **Unlock** in the top-right corner. Then click **Convert.**

Once converted, a module tool becomes a regular scenario﻿, and appears as a scenario﻿ within your **Make AI Agents tools** and in the **All scenarios** folder.

If you only need to edit the module tool (not its scenario), see the [Edit module tool](https://help.make.com/module-tools-for-ai-agents#qLiN5 "Edit module tool ") section below.

You can also rename the module tool from the scenario﻿ diagram. Just click its name, and enter a new one.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/jl4t8WVJ1-GAUdpkvWCm9-20250926-164755.png "Document image")

﻿

All the scenarios﻿ created for module tools are stored in the **Module tools** folder under the **All** and **Uncategorized** folders. These folders aren't editable, but you can copy the scenarios﻿ into your own folders.

You can identify a scenario﻿ created for module tools by a **Tool** icon next to it.

In the **Module tools** folder, you can also activate/deactivate the scenario﻿ using the toggle. Click the three dots to edit the custom properties of the scenario﻿, move it to a folder (it will still be visible in the **Module tools** folder), clone it, or delete it.

![Document image](https://app.archbee.com/api/optimize/PL8X94efBsjvhfQV3wyyj-UO6_svAryRxIoqls4mqPq-20250822-112452.png "Document image")

﻿

### Edit a module tool

You can edit a module tool in the **AI Agents** configuration tab or in the **Make AI Agents > Run an agent** module.

To edit a module tool in the **AI Agents** configuration tab:

1

Go to the **AI Agents** configuration tab on the left sidebar.

2

Select your agent from the list.

3

In the **Tools** section, click the gear icon next to the module tool you want to edit.

![Document image](https://images.archbee.com/PL8X94efBsjvhfQV3wyyj-6ZVM7qcath-jJ_XS-rG9T-20250822-113048.png?format=webp "Document image")

﻿

4

Edit the module tool in the **Update tool** window, and click **Save**.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/dY4glJFk4QoGw2AIJI63p-20250926-162642.png "Document image")

﻿

To edit a module tool in the **Make AI Agents > Run an agent** module:

1

Open the **Make AI Agents > Run an agent** module in a scenario﻿.

2

In the **Agent** field, select your agent**.**

3

Click the gear icon next to the module tool you want to edit.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/_97HZnXm2-DpOM_zbqSjn-20250924-075931.png "Document image")

﻿

4

Edit the module tool in the **Update tool** window, and click **Save**.

Your module tools and scenarios created for them will be edited.

### Delete a module tool

You can delete a module tool in the **AI Agents** configuration tab or in the **Make AI Agents > Run an agent** module.

To delete module tools in the **AI Agents** configuration tab:

1

Go to the **AI Agents** tab in the left sidebar.

2

Select your agent from the list.

3

In the **Tools** section, click the delete icon next to the module tool you want to delete.

![Document image](https://images.archbee.com/PL8X94efBsjvhfQV3wyyj-GN8lCqaAT9RAAMkuvcDdN-20250822-113954.png?format=webp "Document image")

﻿

4

Click **Delete** to confirm.

To delete a module tool in the **Make AI Agents > Run an agent** module:

1

Open the **Make AI Agents > Run an agent** module in a scenario﻿.

2

In the **Agent** field, select your agent.

3

Click the delete icon next to the module tool you want to delete.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/a0CChd4iK_XJEPGQRa5Be-20250924-081021.png "Document image")

﻿

4

Click **Delete** to confirm.

In both cases, your module tool will be deleted **from all instances of the agent** across all scenarios﻿. However, the module tool's scenario﻿ remains in your **Module tools** folder. So, you can add it again later.

To delete a module tool from your account, you need to delete its scenario﻿:

1

Go to the **Module tools** folder.

2

Click the three dots next to the module tool's scenario﻿, then **Delete**.

![Document image](https://app.archbee.com/api/optimize/PL8X94efBsjvhfQV3wyyj-bQrRXgiFjLZtRXkZbpRBW-20250822-114518.png "Document image")

﻿

3

Click **Delete** once again.

This permanently deletes the scenario﻿. Once deleted, you can't use that module tool again.
