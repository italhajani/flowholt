# API key

1 min

To add an API key to your account:

1

Click your profile icon in the upper right corner and select **Profile**.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/NlBXGPBlWpls96yTFhtkX-20251002-092925.png?format=webp "Document image")

﻿

2

Click the **API access** tab, and then the **Add token** button.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/gKKcKV_1JUo00tPeEbasF-20251002-093926.png "Document image")

﻿

3

Select the necessary **Scopes** andenter a **Label** for your token.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/HhoZ_ojMk7Wm3vaPOAPez-20251002-102205.png "Document image")

﻿

You can review the scopes in the [﻿Make﻿ API documentation](https://developers.make.com/api-documentation/authentication/api-scopes-overview "https://developers.make.com/api-documentation/authentication/api-scopes-overview").

4

Click the **Add** button to confirm your selections.

5

Copy your new API key to your clipboard and store it in a safe place. **This is the only time when you can see the whole API key.** The API key is required to set up the VS code extension.

Your Make﻿ API keys are listed in your profile in the **API Access** tab where you can view and manage your API keys.
