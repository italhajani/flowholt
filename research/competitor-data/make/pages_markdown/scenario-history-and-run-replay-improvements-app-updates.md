# Scenario history and run replay improvements, app updates

7 min

## Scenario run replay improvements

**Replay runs from Run detail:** You can now replay a scenario﻿ run directly from the [Run detail](https://help.make.com/scenario-run-replay#JRQkH "Run detail") page, giving you another way to replay runs without navigating back to the Scenario﻿ Builder.

**View replayed runs in detail:** After you replay a scenario﻿ run, click [View run](https://help.make.com/scenario-run-replay#whEDm "View run") to open its run details in a new tab. You can see the replayed run without leaving your current view.

## Scenario history columns

The Scenario﻿ history now supports [showing and hiding columns](https://help.make.com/scenario-history#GiUD1 "showing and hiding columns"). This gives you more control over what you see in the Scenario﻿ history.
