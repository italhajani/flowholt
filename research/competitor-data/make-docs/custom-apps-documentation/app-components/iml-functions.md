# Custom IML functions

IML functions is a powerful feature that allows you to write your own JavaScript functions and execute them inside IML expressions to process data.

Before getting started with custom IML functions, learn more about the the Integromat Markup Language (IML) .

Custom IML functions are not available by default. Contact us via our helpdesk arrow-up-right if you need to create a custom IML function.

All built-in IML functions and your custom IML functions are available for you to use inside your own custom functions. You can access them in the iml namespace like this: iml.parseDate() .

```
iml
```

```
iml.parseDate()
```

A list of built-in IML functions is available here arrow-up-right .

Only JavaScript built-in objects and Buffer are available for you to use. You can use all features of ES 6, like arrow functions, destructuring, etc…
