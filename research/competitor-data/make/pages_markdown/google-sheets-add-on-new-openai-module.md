# Google Sheets add-on, new OpenAI module

5 min

### New app

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/OTBb1d9wjsRlEGlsDdTZ0_cropped-adp-workforce-now.png?format=webp?format=webp "Document image")

﻿

﻿[ADP Workforce Now](https://apps.make.com/adp-workforce-now "ADP Workforce Now") — This new HR management app lets you access job applications, worker data, job requisitions, and onboarded applicants from your ADP account. It helps you bring essential HR data into your scenarios and keep them aligned. The ADP Workforce Now app is only available on the [Make Enterprise plan](https://www.make.com/en/enterprise "Make Enterprise plan").

### Updated apps

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/L6MT49gFwxtpkTbxO0Dqb_cropped-openai-gpt-3.png?format=webp "Document image")

﻿

﻿[OpenAI](https://apps.make.com/openai-gpt-3 "OpenAI") — We’ve added a new module named **Download a Container** **file**, which lets you retrieve container files using their IDs.

The **Create a model response** module now includes a new **Tool choice** option: **Code interpreter**.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/ARE6lBnw39qe_OUF5yvuB_google-sheets.png?format=webp "Document image")

﻿

﻿[Google Sheets](https://apps.make.com/google-sheets "Google Sheets") — The Make add-on for Google Sheets is back. You can install it from the [Google Workspace Marketplace](https://workspace.google.com/marketplace/app/make/947306521495?flow_type=2 "Google Workspace Marketplace") to connect your spreadsheets with Make, run scenarios, and send or receive data without leaving Google Sheets.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/o1NZQUX6QYwx1WF9AavFd_cropped-microsoft-teams.png?format=webp?format=webp "Document image")

﻿

﻿[Microsoft Teams](https://apps.make.com/microsoft-teams "Microsoft Teams") — The **Send a message** module now includes an **Attachments** field, allowing you to add attachments using OneDrive links.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/jUf51s89GtVbx1RS_bNln_image.png?format=webp "Document image")

﻿

﻿[Monday.com](https://apps.make.com/monday "Monday.com") — This app now includes two modules for creating items: **Create an Item** (by board ID) and **Create an Item (advanced)**, which offers an alternative setup for item creation.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/OIowiPUEnIQfyUwEJbqvo_runway.png?format=webp "Document image")

﻿

﻿[Runway](https://apps.make.com/runway-ml-api "Runway") — We’ve added a new module, **Generate an Image**, that creates images from a text prompt and optional reference images.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/Pl1O6_0qP9kKN0IlPueOf_trello.png?format=webp "Document image")

﻿

﻿[Trello](https://apps.make.com/trello "Trello") — The **Watch Boards** module has been renamed to Watch **Boards Created by Me** to better reflect its functionality.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/6WcF21beVeYucad2e99GE_klaviyo.png?format=webp "Document image")

﻿

﻿[Klaviyo](https://apps.make.com/klaviyo "Klaviyo") — We've added a new connection type: **Klaviyo OAuth**. The previous API key connection option is now deprecated.
