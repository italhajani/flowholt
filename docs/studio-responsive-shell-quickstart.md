# Studio Responsive Shell Quickstart

This improves the current FlowHolt Studio before the full redesign phase.

## What changed

1. Mobile app navigation is simpler and easier to reach.
2. Studio action buttons wrap better on smaller screens.
3. The canvas tool rail becomes a horizontal quick toolbar on mobile.
4. Small screens now get pane switching between:
   - Canvas
   - Step
   - Link
   - Data
   - JSON
5. The right-side Studio stack is more usable on phones and tablets.

## How to see it

1. Restart `flowholt-web`
2. Open any workflow in `/app/studio/[workflowId]`
3. Shrink the browser width or open device toolbar in browser devtools
4. You should now see:
   - mobile top navigation in the app shell
   - Studio buttons stacking more cleanly
   - a mobile pane switcher above the canvas
   - a horizontal node toolbar instead of a cramped vertical rail

## Easy meaning

Before this, Studio was mostly desktop-first.
Now the current editor shell is much easier to use on smaller screens, even before the final premium redesign survey and rebuild.
