# Make AI Content Extractor, new Shopify version, app updates

2 min

### New apps

| Document image  ﻿ | ﻿[Make AI Content Extractor](https://apps.make.com/make-ai-extractors "Make AI Content Extractor") - We've released a new built-in app that extracts structured text and metadata from files like PDFs, Word documents, images, and audio recordings—directly within your Make scenarios. The app is currently in open beta, available to all customers on paid plans, except for the Enterprise plan at the moment. |
| --- | --- |
| Document image  ﻿ | ﻿[Pictory](https://apps.make.com/pictory "Pictory") - This AI-powered platform enables you to create and edit engaging videos quickly and easily. You can manage transcriptions, summaries, and video generation in your Pictory account using this new app. |
| Document image  ﻿ | ﻿[Postiz](https://apps.make.com/postiz "Postiz") - This open-source social media tool lets you create, update, and delete posts, upload files, and retrieve post data from your Postiz account—directly within your scenarios. |

### Updated apps

| Document image  ﻿ | ﻿[OpenAI](https://apps.make.com/openai-gpt-3 "OpenAI") - We’ve added new parameters to our **Create a Model Response** module and **Create a Completion** module. |
| --- | --- |
| Document image  ﻿ | ﻿[Shopify](https://apps.make.com/shopify "Shopify") – We’ve released an updated version of the Shopify app using the GraphQL API, offering better performance and improved data handling. To learn more, see [Shopify tips and examples](https://apps.make.com/shopify-tips-and-examples "Shopify tips and examples"). |
| Document image  ﻿ | ﻿[Webhooks](https://apps.make.com/gateway "Webhooks") - You can now attach multiple API keys to your custom webhook. This feature provides an extra layer of security to control access management. |
| Document image  ﻿ | ﻿[Make](https://apps.make.com/make "Make") - Connections to the Make app can now be established using OAuth2 or an API key. |
| Document image  ﻿ | ﻿[Salesforce Pardot](https://apps.make.com/salesforce-pardot "Salesforce Pardot") - It’s now possible to connect using OAuth client credentials, streamlining the authentication process. |
| Document image  ﻿ | ﻿[Calendly](https://apps.make.com/calendly "Calendly") - We've added a new module: **Cancel an Event**. |
| Document image  ﻿ | ﻿[LiveChat](https://apps.make.com/livechat "LiveChat") - A new connection procedure has been imlpemented for LiveChat, requiring your client credentials. |
| Document image  ﻿ | ﻿[Printful](https://apps.make.com/printful "Printful") - The API key connection type has been deprecated. You can now create a connection using client credentials. |
