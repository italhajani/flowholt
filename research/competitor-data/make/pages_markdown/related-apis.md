# Related APIs

0 min

The Make API allows you to define scenario﻿ inputs and outputs programmatically, as well as use the API to pass input values to a scenario﻿ and run it.

In the API, the definition of scenario﻿ inputs and outputs is called **scenario interface**.

You can also use the **Run a scenario** module of the Make app.

The following endpoints are available:

* ﻿[Get scenario interface](https://developers.make.com/api-documentation/api-reference/scenarios#scenarios-scenarioid-interface "Get scenario interface")﻿

* ﻿[Update scenario interface](https://developers.make.com/api-documentation/api-reference/scenarios#scenarios-scenarioid-interface-1 "Update scenario interface")﻿

* ﻿[Run a scenario](https://developers.make.com/api-documentation/api-reference/scenarios#scenarios-scenarioid-run "Run a scenario")﻿
