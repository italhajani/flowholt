# Getting started

The Make REST API allows using HTTP requests to access Make data and control the Make platform without opening its graphical interface. This allows you to embed Make features into your software, add features on top of the platform, and automate your tasks that you perform in Make. To use the Make API, you need a Make account. Once logged in, you can create an authentication token or request an OAuth 2.0 client arrow-up-right and start making calls to the API .

The API allows you to interact with multiple Make resources .
