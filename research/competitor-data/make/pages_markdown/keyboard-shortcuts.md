# Keyboard shortcuts

1 min

You can save time when editing scenarios﻿ with keyboard shortcuts.

Make features these shortcuts:

| **Shortcut** | **Action** |
| --- | --- |
| + | Zoom in |
| - | Zoom out |
| **Win:** Ctrl+S  **Mac:** ⌘+S | Save scenario |
| **Win:** Ctrl+C on module  **Mac:** ⌘+C on module | Copy module |
| **Win:** Ctrl+V  **Mac:** ⌘+V | Paste module |
| Space+drag on module | Move all modules |
| **Win:** Ctrl+Enter  **Mac:** ⌘+Enter | Run once |
| Shift+drag | Create a box to select multiple modules |
| **Win:** Shift+A  **Mac:** ⇧+A | Auto-align |
| **Win:** Ctrl+Shift+Enter  **Mac:** ⌘+⇧+Enter | Run with existing data |
