# Context modules added to Make AI Agents app

3 min

Make AI Agents is available in Open Beta to all users on paid plans.

Previously, Make AI Agents required you to manually upload context files in the **AI Agents** configuration tab, making real-time updates to files difficult. Now, you can manage your agent's access to the latest context files directly in a scenario using context modules. Found in the **Make AI Agents** app, context modules allow you to add, delete, and retrieve context files— all within a scenario.

## What's new?

* You can now manage your AI agent's context in a scenario using **Make AI Agents** context modules.

* Add, update, retrieve, and delete context files using the corresponding modules.

* Choose between uploading context as files or plain text.

* Replace outdated context files with new versions.

* Remove specific context or all context files from an agent's memory.

* You can still manually upload context files in the **AI Agents** configuration tab, on the left sidebar.

## Where to find it

You can find context modules in the Scenario Builder:

1. Click on the **Create a new scenario** button in the top right-hand corner, which takes you to the Scenario Builder.

2. In the app search next to the giant plus, type and select **Make AI Agent**.

3. Under the **Context** category, you can see all context modules.

## **Learn more**

To find out more about Make AI Agents context modules and their settings, head to [Make AI agent reference](YuE3-2TSsMXUX1SWpT8ep)﻿.
