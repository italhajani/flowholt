# Install the Make CLI

The Make CLI is available for macOS, Linux, and Windows. Choose the installation method that suits your setup.

### hashtag Install via npm

To install via npm, install Node.js arrow-up-right and run this command in your terminal:

```
npminstall-g@makehq/cli
```

Or to run directly without installing:

```
npx@makehq/cli<command>
```

Example

```
npx@makehq/cliscenarioslist--team-id=123
```

### hashtag Install via Homebrew (macOS and Linux)

To install via Homebrew, run this command in your terminal:

```
brewinstallmake-cliintegromat/tap/make-cli
```
