# Incomplete executions retry

2 min

## Improvements and changes

Users can now retry multiple incomplete executions at once. You can either select specific incomplete executions or attempt to retry all of them. You can see approximately when an incomplete execution will be rerun in the scenario﻿ incomplete executions tab.

## Apps updates

We've improved the way that custom app logos are processed for more accurate representation. For more details on how single color, multicolor, and grayscale logos are processed, view the specifications and examples [here](https://developers.make.com/custom-apps-documentation/app-logo#app-logo "here").
