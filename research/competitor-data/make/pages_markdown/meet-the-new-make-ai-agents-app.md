# Meet the new Make AI Agents app

3 min

We’re excited to introduce the **new Make AI Agents app:** a major improvement to Make AI Agents designed to make building and testing AI agents in Make﻿ **faster, more transparent, and more intuitive than ever**.

This new version brings AI agents directly to the Scenario Builder with a visual, end-to-end experience. You can now build, run, test, and understand your agents in one place with full visibility into how they make decisions.

To get started, create a scenario and add the **Make AI Agents (New)** > **Run an agent** **(New)** module to the canvas:

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/QwygkDJoNWxPUTJe7KBIO-20260213-162009.png?format=webp "Document image")

﻿

This release introduces a more efficient and transparent way to work with Make AI Agents, with improvements across the entire building and testing flow:

* **See agents in action**Watch real-time animations that show when your agent is thinking, how long it takes, and which tools it runs in the exact order they’re called.

![Document image](https://archbee-image-uploads.s3.amazonaws.com/oAyFj2GHlBeBVWF5OAir2/HB7lY_QFhJ3AS1i5aabOj-20260213-162009.gif "Document image")

﻿

* **Build, test, and debug in one place**Create agents, add tools and knowledge, chat with them, test scenarios, and debug issues directly inside the Scenario Builder without switching tabs.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/ieT9D3NOobIqHNzWMFGIt-20260213-162009.png?format=webp "Document image")

﻿

* **Understand every decision with agent reasoning**Inspect the agent’s step-by-step logic in the **Reasoning** panel to see why it chose a certain action, making troubleshooting and optimization easier.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/1zldJn86KpQu1mgZwrLJ6-20260213-162008.png?format=webp "Document image")

﻿

* **AI-assisted tool configuration**Let the agent populate tool fields for you, enabling faster setup, more flexible routing, and easier handling of unstructured data.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/UNCc7DrWD_a0alFPF3qIX-20260213-162009.png?format=webp "Document image")

﻿

* **Flexible file handling and knowledge**Give agents access to files by uploading them to the knowledge base or by mapping file data directly as inputs.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/SYgtTc8tT2ifsBBsNFISo-20260213-162009.png?format=webp "Document image")

﻿

* **Share agents with ease**Generate public links to AI agent scenarios so others can reuse or build on your work, reducing setup time and duplication across teams.

![Document image](https://images.archbee.com/oAyFj2GHlBeBVWF5OAir2/a2RPDHgoCI0b0fCPY5Qj9-20260213-162009.png?format=webp "Document image")

﻿
