# MCP tools for AI agents

2 min

This page reflects a **previous version of Make AI Agents**. For the latest information, see [Make AI Agents (New)](/make-ai-agents-new)﻿.

You can connect your AI agent to MCP servers to access additional tools.

To add an MCP tool for an AI agent:

1

Go to the **AI Agents** configuration tab on the left sidebar.

2

Select an agent where you want to add an MCP tool.

3

In the **MCP** section, click **Add**.

![Document image](https://images.archbee.com/PL8X94efBsjvhfQV3wyyj-9NSw3Mw-AULKzXjRzkv2q-20250826-135709.png?format=webp "Document image")

﻿

4

Create a connection to an available MCP server or choose an existing connection.

See [here](https://apps.make.com/mcp-client#vHyR5 "here") how to create a connection to an MCP client.

5

Toggle on the tools that you want the agent to access.

6

Click **Save**.

![Document image](https://images.archbee.com/PL8X94efBsjvhfQV3wyyj-7uGD8E_mWot8Ovwuad2h1-20250826-140010.png?format=webp "Document image")

﻿

7

Click **Save** in the top-right corner to save the changes in your AI Agent.

![Document image](https://app.archbee.com/api/optimize/oAyFj2GHlBeBVWF5OAir2/AK3jIOIL3REuFjVJW6T4h-20250929-071225.png "Document image")

﻿

You can now use your MCP servers and their tools in your AI agent. You can also:

* Click the drop-down to enable and disable active tools

* Click the gear icon to edit the connection

* Click the delete icon to delete the MCP tool from your AI agent

![Document image](https://app.archbee.com/api/optimize/PL8X94efBsjvhfQV3wyyj-ZWp7NSwEdD_vZdKlGQapd-20250826-141225.png "Document image")

﻿

﻿

Within the **Make AI Agents > Run an agent** module, you can click the drop-down to check all the active MCP tools added to your AI agent.

![Document image](https://images.archbee.com/PL8X94efBsjvhfQV3wyyj-KoWOp5pk_7YEhRK7zw3TV-20250826-142058.png?format=webp "Document image")
