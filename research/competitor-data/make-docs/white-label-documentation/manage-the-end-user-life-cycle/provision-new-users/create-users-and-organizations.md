# Create users and organizations

Although you can add users and create organizations in the Administration interface, using API lets you automate your provisioning. You can create Make scenarios using the HTTP arrow-up-right and Make modules arrow-up-right to perform these tasks. The above API endpoints let you perform all necessary provisioning tasks for new users.

Although SSO supports JIT provisioning to create users and organizations, you still need to assign permission roles and define the organization's license.
