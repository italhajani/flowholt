# FlowHolt

FlowHolt is a workflow automation platform with a dashboard, studio, templates, executions, integrations, vault management, and a zero-budget FastAPI backend.

## Frontend

- Vite
- React
- TypeScript
- Tailwind CSS

## Backend

- FastAPI
- SQLite for local-first persistence
- Optional local-model execution with Ollama

## Run locally

```powershell
npm install
npm run dev
```

In another terminal:

```powershell
npm run backend:dev
```
