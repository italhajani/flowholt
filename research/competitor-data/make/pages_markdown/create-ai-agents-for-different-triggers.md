# Create AI agents for different triggers

1 min

Create AI agent scenarios using common triggers: chat messages, emails, forms, webhooks, and mailhooks. Use the guided examples to build new AI agents or troubleshoot existing ones.

Once done, you'll have the basic scenario﻿ structure. Tailor it to your needs by adding agent tools, knowledge, and specific instructions.

**Before you begin:** A trigger module is the first module in the scenario. It determines how the agent receives its requests or information.
